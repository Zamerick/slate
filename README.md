# Slate

## Description
As in "Clean Slate". This is a pre-configured VueJS project for starting new projects. The current and future state of this project reflect my personal preferences when it comes to building vue apps. Questions are always welcome.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```

### Run your unit tests
```
yarn run test:unit
```
